# 3 到 5 分钟演示视频脚本

## 一、开场

大家好，我展示的是 Stage Outfit Match Agent，中文名是演出服搭配智能体。这个系统面向舞蹈演出、校园晚会和比赛舞台，可以根据舞蹈题目或视频链接，自动推荐完整演出服搭配。

## 二、展示公开访问链接

这里是系统的公开访问页面。后端部署在 Render，前端部署在 Vercel 或 GitHub Pages，因此老师可以直接通过链接访问。

## 三、输入示例问题

我在输入框中输入：

```text
我想跳 NewJeans《Super Shy》，想要适合学校舞台的女团风演出服。
```

接着选择风格按钮“甜美”，预算选择“中预算”。

## 四、点击生成

点击“生成五套搭配”。系统开始运行，可以看到右侧日志依次显示：

1. 正在读取舞蹈信息
2. 正在匹配舞台风格
3. 正在生成五套造型
4. 已补充商品搜索链接
5. 已生成 PDF 搭配报告

## 五、展示输出结果

生成结果包含舞蹈风格分析、参考来源和风格关键词。下面是五套搭配卡片，每套都有搭配照片、上衣、下装、鞋子、配饰、单品价格、总价和搜索链接。

系统会自动按照价格从低到高排序，方便用户根据预算选择。

## 六、展示链接

我点击其中一个单品的淘宝、京东或拼多多搜索链接，可以看到系统已经根据单品关键词生成可购买方向的搜索结果。

## 七、下载 PDF 报告

点击“下载 PDF 报告”，系统会生成一份完整搭配报告。报告中包含输入信息、舞蹈分析、参考来源、五套搭配明细和试穿建议。

## 八、总结

这个智能体满足作业要求：

1. 可以公开访问
2. 使用公开大模型 API，也支持 mock 模式演示
3. 通过提示词工程和轻量工具调用完成搭配推荐
4. 能解决一个具体问题：舞蹈演出服怎么搭配
5. 能输出完整 PDF 报告
6. 整个过程适合录制 MP4 演示视频
